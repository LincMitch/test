# HxUI Sketch Plugin

## How to develop
First [clone the repo](http://github.com/airbnb/react-sketchapp):
Then clone this repo as a submodule

Install the dependencies
```
npm install
```

Run with live reloading in Sketch
```
npm run render
```

Or, to install as a Sketch plugin:
```
npm run build
npm run link-plugin
```
